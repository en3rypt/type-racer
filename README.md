
# typ.io
Type Race game using express.js

## Run Locally

Clone the project

```bash
  git clone git@github.com:en3rypt/type-racer.git
```

Go to the project directory

```bash
  cd type-racer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
